게임 방법 익히기 (연습 플레이)

⚓ 직접 해보며 배우는 게임

읽기만 하면 헷갈려요. 짧은 연습 항해로 한 라운드를 진짜처럼 플레이해 보세요. 버튼만 눌러 따라가면 2~3분 안에 룰이 손에 익어요.

📖 글로 된 규칙 전체 보기 (선생님·참고용)

한 라운드: ① 폭풍 예보(세기 일부만 공개) → ② 협상(약속은 자유, 안 지켜도 됨) → ③ 비밀 기여(각자 몰래 제출, 합계·기준 미달자 수만 공개) → ④ 판정(숨은 세기 공개 후 합계와 비교) → ⑤ 정산(남긴 자원이 내 점수).

🛡 합계 ≥ 폭풍 → 도시 방어 성공(남는 자원은 파도에 쓸려감). 💥 합계 < 폭풍 → 피해 1 (폭풍의 3/4도 못 막으면 피해 2).

항구 기준: 매 라운드 제시되는 개인 기준. 방어 성공 + 기준 이상 기여한 최다 기여자는 명예 토큰(+2점). 방어 성공 + 모둠원 80% 이상이 기준 이상이면 방파제 강화 +1(강화 2개는 실패 시 투표로 사용).

승패: 피해 3 → 도시 함락, 모둠 전원 패배. 끝까지 버티면 최종 피해 보정(피해 0 → +6점, 1 → +3점, 2 → 0점) 후 점수 1등이 우승. 점수 = 남긴 자원 + 명예 토큰 + 생존 보정.

선생님: 단계 전환은 교사 화면의 "다음 단계" 버튼으로 진행돼요. 권장 모둠당 4~6명 · 6라운드. 디브리핑 질문 — 왜 세금은 자발적 기부가 아니라 강제일까? 기여가 익명이 아니었다면 우리 모둠은 어떻게 달라졌을까?

연결이 불안정해요. 다시 연결하는 중…
